# WPF Dynamic GUI Fields

In diesem kleinen Projekt wird gezeigt, wie für eine GUI die Eingabecontrols (hier TextBox) dynamisch einer GUI hinzugefügt werden kann.

<img src="MainWindow.png" style="width:750px;"/></br>

Durch Klick auf den Button **Weiteres Feld** wird eine neue Zeile mit einem Label und Eingabefeld erstellt.

<img src="MainWindow_A.png" style="width:750px;"/></br>

Über einen Template Selektor wird in Abhängigkeit des Datentyp ein entsprechendes Eingabe Control erstellt.
